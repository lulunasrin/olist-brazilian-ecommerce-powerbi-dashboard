# Dataset

This project uses the **Brazilian E-Commerce Public Dataset by Olist**.

**Source:** Kaggle

🔗 https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

## Dataset Description

The dataset contains approximately 100,000 orders placed between 2016 and 2018 and includes:

- Customers
- Orders
- Order Items
- Products
- Payments
- Reviews
